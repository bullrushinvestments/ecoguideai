# EcoGuideAI

A personalized climate tech solution that leverages AI to provide small businesses with actionable eco-strategies and content resources.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
